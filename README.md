### Usage
```
Usage of run:
  -allow-origin string
    	Allow specified origin (or * for all origins) to make requests to this server
  -cache string
    	wasm cache dir (default "cache")
  -http string
    	HTTP bind address to serve (default ":8080")
  -v	print verbose information
```
